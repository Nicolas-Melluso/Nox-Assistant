# Guía rápida v0.4.2

## Integración de APIs externas
- Usa `ExternalAPIClient` para conectar servicios REST, IoT, etc.
- Integra el cliente en CoreEngine y expón la funcionalidad vía CLI, API o SPA.
- Consulta los ejemplos en README y tests.

## Gestión de feature flags
- Los flags se configuran en `src/config/feature_flags.yaml`.
- Puedes editarlos desde CLI (menú interactivo), API REST o SPA visual.
- Los cambios se reflejan en todas las interfaces.
