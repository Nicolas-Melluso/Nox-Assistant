# Documentación Custom Voice Assistant

## Descripción

Este proyecto implementa un asistente de voz personalizado para Windows, capaz de reconocer intenciones en español y ejecutar acciones reales. Utiliza modelos de ML para clasificación de intenciones y reglas robustas para enrutamiento.

## Estructura de carpetas

- `src/` - Código fuente principal
- `scripts/` - Scripts de automatización y entrenamiento
- `training/datasets/raw/` - Datasets crudos versionados
- `results/` - Resultados y logs de ejecución
- `agent-history/` - Historial de sesiones (no versionado)

## Cómo contribuir

1. Clona el repositorio
2. Instala dependencias (`requirements.txt`)
3. Ejecuta los scripts de entrenamiento o prueba
4. Sube solo cambios en código y datasets crudos
