# Template para Actualizaciones de Agent History

Usa este template cuando quieras actualizar el AGENT_LOG.md con nueva información.

## Estructura Recomendada

```markdown
## Session X (DD-MM-YYYY)

### Objetivos de esta sesión
- Punto 1
- Punto 2

### Qué hicimos
- Acción 1
- Acción 2

### Resultados
- Resultado 1
- Resultado 2

### Problemas encontrados
- Problema 1
  - Solución aplicada
- Problema 2
  - Status: En espera

### Aprendizajes
- Insight 1
- Insight 2

### Próximos pasos
- [ ] Tarea 1
- [ ] Tarea 2

### Notas para el siguiente Agent
- Nota importante 1
- Nota importante 2
```

## Información a registrar siempre

- **Fecha**: DD-MM-YYYY
- **Cambios en el roadmap**: Si hay
- **Decisiones técnicas**: Por qué elegimos X sobre Y
- **Bloqueadores**: Qué nos frenó
- **Éxitos**: Qué funcionó bien
- **Código agregado**: Referencia a archivos nuevos

## Ejemplo completo

```markdown
## Session 1 (17-04-2026)

### Objetivos
- Estructurar proyecto desde cero
- Crear Agent History system
- Definir roadmap

### Qué hicimos
- Creamos estructura de carpetas
- Definimos 4 fases del proyecto
- Implementamos Agent History

### Próximos pasos
- [ ] Crear dataset inicial de intenciones
- [ ] Desarrollar data_pipeline.py
```
